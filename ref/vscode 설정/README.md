## 확장 프로그램 사용법

### Setting Sync

* Upload your settings
	
	```
	Shift + Alt + U를 누르고 팔레트 창에 위에서 만든 토큰을 입력하고 Enter
	```
	
* Download your settings to other devices

	```
	Shift + Alt + D를 누르고 팔레트 창에 위에서 획득한 Gist ID를 입력한 다음 Enter
	```
